== Description ==

Smartarget FAQ helps your users to get most common questions answered in the most visible way by showing them on every page on your website.
Answer the most relevant question and make it visible for users so they gain more trust in you. For example:
What payment options do you accept?
What is your return policy? *Customer support working hours?

== Installation ==

1. Unpack plugin zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open plugin settings page
4. Configure settings and press Save
